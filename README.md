# Machine Learning for Financial Market Prediction

This project implements a **machine learning pipeline to predict short-term stock market direction** using financial indicators. The goal is to create a simple quantitative trading strategy based on historical market data.

## Objective

- Predict whether the market (S&P500 ETF - SPY) will move up or down the next day.  
- Evaluate a machine learning-based trading strategy against the market benchmark.  
- Demonstrate feature engineering, ML modeling, and performance metrics.

## Features

The model uses the following financial indicators:

- **MA5**: 5-day moving average  
- **MA20**: 20-day moving average  
- **Volatility**: 20-day rolling standard deviation of returns  
- **Momentum**: 10-day price momentum  
- **Returns**: Daily percentage change
